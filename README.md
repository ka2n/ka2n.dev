# micronextblog


## Configuration

### Environment variables

**MICROCMS_KEY**

API key for microcms.io

e.g.: `XXXXXXX-XXXX-XXXX-XXXX-XXXXXXX`

**MICROCMS_ENDPOINT**

Endpoint for microcms.io

e.g.: `https://<id>.microcms.io/api/v1`
